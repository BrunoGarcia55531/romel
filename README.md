## 🚀 Cómo deployar en Vercel

### Opción 1: Usando Git (Recomendado)

1. **Inicializa Git en tu carpeta:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```

2. **Sube a GitHub:**
   - Crea un repositorio en GitHub
   - Sube tu código:
     ```bash
     git remote add origin https://github.com/tu-usuario/tu-repo.git
     git branch -M main
     git push -u origin main
     ```

3. **Conecta a Vercel:**
   - Ve a [vercel.com](https://vercel.com)
   - Haz click en "New Project"
   - Selecciona tu repositorio de GitHub
   - Haz click en "Deploy"

### Opción 2: Usando Vercel CLI

1. **Instala Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Deploy directamente:**
   ```bash
   cd c:\Users\BRUNO\Desktop\romel
   vercel
   ```

3. **Sigue las instrucciones en terminal**

## 📋 Archivos incluidos

- `index.html` - Página principal (Vercel lo sirve automáticamente)
- `vercel.json` - Configuración de Vercel
- `.gitignore` - Archivos a ignorar en Git

## ⚠️ Importante

- Si tienes `image_0.jpeg`, **asegúrate de subirlo también** a Vercel para que la imagen aparezca
- La imagen debe estar en la misma carpeta que `index.html`
